Data uploader
This is my project that allows users to upload CSV files and display them in a graph view.

Under the hood, an SQLite database is used, along with Flask, Peewee and Python in the backend. In the frontend, plain HTML, CSS, and JavaScript are used. Libraries used in the frontend include jQuery and d3.js.

The client communicates via POST/GET requests, initially loading all previously uploaded files. The database uses one table to store the files.

To get started, please upload CSV files on the right-hand side using the tabular view. When an entry is uploaded, you can delete or display the file. Clicking the display button will redirect you to a second HTML page with a graph displaying the contents of the CSV file.
